# E-Commerce Platform - Authentication Frontend

This is a professional, modern React frontend for the authentication pages (Login & Signup) of an e-commerce platform.

## Features
- Minimal, responsive design using React functional components
- User Login and Signup with validation
- Clean, modern CSS
- Integration-ready with backend API

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open your browser at [http://localhost:3000](http://localhost:3000)
