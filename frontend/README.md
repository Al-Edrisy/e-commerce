# E-Commerce Frontend

This is a modern e-commerce frontend application built with React, TypeScript, and Vite.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **State Management**: React Context + TanStack Query
- **Routing**: React Router DOM
- **Forms**: React Hook Form + Zod

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Features

- **Authentication**: Login, Register, Google OAuth (Mock), Role-based access (Admin/Customer)
- **Product Management**: Browse, Search, Filter, Sort, Quick View
- **Shopping**: Cart, Wishlist, Checkout with payment simulation
- **Admin Panel**: Dashboard, Product/Order/User management
- **Responsive Design**: Mobile-friendly UI with dark mode support
